# Bank Transaction Analysis System

## Overview
A Python-based financial analytics system that analyzes 900+ bank transactions using exploratory data analysis and machine learning techniques.

## Features
- Transaction classification and preprocessing
- Category-wise spending analysis
- Monthly trend detection
- Recurring merchant identification
- K-Means clustering for spending segmentation
- Isolation Forest anomaly detection for fraud detection
- Data visualization using Matplotlib and Seaborn

## Tech Stack
- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib
- Seaborn
